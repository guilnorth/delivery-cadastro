### Instalação Local

#### Pré requisitos

- Node e React Cli
- NPM/Yarn

Após clonar o repositório, dentro da pasta do projeto execute:

```sh
# Através do yarn
yarn install
yarn start

# Ou com npm
npm install
npm start
```
